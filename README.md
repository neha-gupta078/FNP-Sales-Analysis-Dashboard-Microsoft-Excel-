Overview
Analyzed sales data for Ferns N Petals (FNP) to identify revenue trends, top products, and festival-driven demand patterns using Microsoft Excel.
Key Insights
Identified top revenue-generating product categories and cities
Detected sales spikes during Diwali, Valentine's Day, and Raksha Bandhan
Analyzed customer demographics to understand buying patterns
Tools Used: Microsoft Excel — Pivot Tables, XLOOKUP, VLOOKUP, Charts, Conditional Formatting, Data Validation
Dashboard Features
Interactive Pivot Table-based dashboard
Festival-wise and city-wise sales breakdown
Monthly trend analysis with line charts
