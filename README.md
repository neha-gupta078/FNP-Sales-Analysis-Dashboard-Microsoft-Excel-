Overview: Analyzed sales data for Ferns N Petals (FNP) to identify revenue trends, top products, and festival-driven demand patterns using Microsoft Excel.

Key Insights
1.Identified top revenue-generating product categories and cities
2.Detected sales spikes during Diwali, Valentine's Day, and Raksha Bandhan
3.Analyzed customer demographics to understand buying patterns

Tools Used: Microsoft Excel — Pivot Tables, XLOOKUP, VLOOKUP, Charts, Conditional Formatting, Data Validation

Dashboard Features
1.Interactive Pivot Table-based dashboard
2.Festival-wise and city-wise sales breakdown
3.Monthly trend analysis with line charts
